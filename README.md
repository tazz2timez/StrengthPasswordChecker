# 🔐 Password Strength Checker (Python)

A simple and effective Python program that checks the strength of a password based on common security rules. It analyzes your password and provides feedback to help you create a stronger, more secure one.

---

## 🚀 Features

- Checks password length (minimum 8 characters)
- Detects:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Classifies password strength:
  - Weak
  - Medium
  - Strong
- Provides helpful suggestions for improvement

---

## 🛠️ Technologies Used

- Python 3
- Regular Expressions (`re` module)
